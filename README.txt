---CHECKPOINT 3 - PROGRAMAÇÃO IMPERATIVA---

Para executar os comandos, seguir o seguinte passo a passo. 


Executar inicialmente o comando: "node Course.js". 

Ao executar, irá listar o nome dos alunos e se eles foram aprovados ou não, com base em suas notas e faltas. 
Caso queira verificar todos os dados dos alunos cadastrados (nome, faltas, notas), basta descomentar a linha 43 (console.log(students);) do arquivo "Course.js" 

Ao executar os outros arquivos (Student.js e Students.js) nada irá aparecer, pois o único comando de listar está no arquivo "Course.js".

Construi o código e nome dos arquivos todos em inglês por questões de boas práticas e costume.
